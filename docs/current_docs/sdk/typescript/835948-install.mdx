# Installation

:::note
The Dagger Typescript SDK requires [Node.js 16.x or later](https://nodejs.org/en/download/).
:::

Install the Dagger Typescript SDK in your project using `npm` or `yarn`:

<Tabs>
<TabItem value="npm">

```shell
npm install @dagger.io/dagger@latest --save-dev
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add @dagger.io/dagger --dev
```

</TabItem>
</Tabs>
